CARTELLA CONTRIBUENTE

 

GESTIONE RICHIESTE

La funzione consente di rispondere alle richieste di predisposizione della cartella contribuente o in generale di chiarimenti circa la situazione fiscale  del soggetto come risultante presso l’Amministrazione Comunale .

La risposta può essere costituita da:

  • un semplice messaggio testuale;
  • un file formato PDF contenenti i dati inerenti la situazione contributiva

 

Il. mezzo attraverso cui viene evasa la richiesta può essere:

  • consegna a mano
  • e-mail

L’evasione tramite e-mail è possibile solo qualora, all’atto della richiesta, sia stato indicato un indirizzo e-mail a cui effettuare la spedizione. Per effettuare l’invio tramite e-mail è necessario spuntare l’apposita casella.

 

Prepara PDF

Questo link è presente quando la richiesta non è stata evasa. Per effettuare la preparazione del file PDF (prima preparazione o sostituzione del file esistente), cliccare in corrispondenza di “Prepara PDF”: verrà aperta, in una pop-up, la pagina di visualizzazione dei dati tributari in cui sarà possibile verificare le informazioni e produrre il file.

 

Visualizza PDF

Questo link è presente quando la richiesta  non è ancora stata evasa ma è già stato preparato il file PDF corrispondente alla cartella tributaria. Tramite esso è possibile visualizzare il file PDF creato.

 

Cancella PDF

Questo link è presente quando la richiesta  non è ancora stata evasa ma è già stato preparato il file PDF corrispondente alla cartella tributaria. Tramite esso è possibile eliminare il file PDF creato.

 

 

NOTA BENE

Se viene effettuata la preparazione del file PDF ma non viene effettuata l’esplicita evasione della richiesta, al successivo accesso alla funzione di gestione della  richiesta il file prodotto può essere visualizzato, sostituito o cancellato.

 

EVASIONE DELLA RICHIESTA

 

L’evasione della richiesta avviene al clic sul pulsante “Salva”: a questo punto il file prodotto non può più essere né eliminato né sostituito . Tutti i  dati relativi alla richiesta, ivi compreso il file PDF corrispondente alla cartella tributaria, sono consultabili attraverso la funzione di esame delle richiesat (scelta di menù ESAMINA RICHIESTE)
