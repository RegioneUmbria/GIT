PRODUZIONE FILE PDF

 

Nella pagina di visualizzazione dei dati del soggetto è disponibile un pulsante “PRODUCI PDF” che consente la creazione del file PDF contenente le informazioni tributarie visualizzate a video.

 

Il pulsante determina l’apertura di una pop-up in cui è possibile profilare la cartella: la profilazione è inerenti alle tipologie di dati che saranno esportati nel file PDF, in relazione alla banca dati considerata. Si può pertanto includere/escludere nella cartella PDF:

  • dati ICI
  • dati TARSU
  • dati COSAP
  • dati CNC
  • dati REDDITI

 

Se l’accesso alla pagina è avvenuto tramite la funzionalità di preparazione della cartella, allora contestualmente alla produzione del file PDF il sistema effettua l’evasione della richiesta stessa.

I dettagli della richiesta, ivi compreso il file creato, potranno essere consultati tramite la funzione di esame richieste.

Il file PDF potrà dunque essere riprodotto a video e stampato ma rimane associato alla richiesta e non potrà più essere sostituito né cancellato: se necessario sarà possibile effettuare una nuova richiesta per lo stesso soggetto.

 

Se invece l’accesso alla pagina è avvenuto tramite la funzionalità di  gestione della richiesta, allora l’evasione della richiesta dovrà avvenire previa espressa conferma da parte dell’utente

Il file PDF creato rimane associato alla richiesta: in questo caso esso, accedendo alla richiesta tramite la funzione di Gestione Richiesta (cui si accede tramite la scelta di menù Evasione Richieste) potrà essere sostituito o anche cancellato, fintantoché non avviene l’esplicita evasione della richiesta da parte dell’operatore.
